FAQ

What is Trackget?

Trackget is a budget tracking web application that allows users to track their income, expenses, and savings goals. Users can input their income, expenses, and savings goals, and Trackget will generate charts that is useful to the user on the Trackget dashboard.

Does this website store my data?

No. Data provided by users is not stored in any kind of database server. However, your local browser (Chrome, Microsoft Edge etc.) remembers your input for data visualization (generating charts). User’s data that are stored on browser is automatically erased upon closing the browser.

Where is this website hosted?

This website is hosted on Pythonanywhere.

I want to send feedback. Where can I do so?

You can send feedback, queries, or any other inquiries related to the website to trackgetoffical@gmail.com.

Why can’t I set my expenses higher than my monthly salary?

Because you shouldn’t spend more than you earn! :)
